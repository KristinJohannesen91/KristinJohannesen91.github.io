Data Analysis Tool

This project was developed to analyze customer data and provide actionable insights. It includes functionalities for data cleaning, visualization, and reporting.

Mathematical Model

This project uses linear regression to model customer spending patterns. The regression equation is: $$ y = mx + b $$

In matrix form, it is represented as: $$ Y = X \\beta + \\epsilon $$
